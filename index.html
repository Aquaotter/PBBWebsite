What is PrisonButBad?

PrisonButBad is a Minecraft server hosted on Minehut that offers a unique and challenging prison experience.

PrisonButBad is also what was supposed to be a 1-week-long expirement server that got way out of hand and is now an official SCP /j

Join the Fun!

To join, simply launch Minecraft Java Edition, select Multiplayer, then click "Add Server" and enter the following details:

  • Server Name: PrisonButBad
  • Server Address: prisonbutbad.minehut.gg
  • The server also works on Bedrock Edition, as we are hosted on minehut!

Once you're in, start your adventure and climb the ranks to become the Warden!
